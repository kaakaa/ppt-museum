# ppt-museum

**pp-museum** is a uploader for .ppt/.pptx/.pdf.

## Usage

```
$ git clone https://github.com/kaakaa/ppt-museum.git
$ cd ppt-museum

// build web application
$ ./gradlew :ppt-museum:installDist

// start application
$ docker-compose up -d
```
And you access to `http://localhost:4567/`

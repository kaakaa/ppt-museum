# ppt-museum

**ppt-museum** is a uploader for .ppt/.pptx/.pdf.


![top](https://github.com/kaakaa/ppt-museum/raw/master/image.png)

## Usage

```
$ git clone https://github.com/kaakaa/ppt-museum.git
$ cd ppt-museum

// start application
$ docker-compose up -d
```
And you access to `http://localhost:4567/`
